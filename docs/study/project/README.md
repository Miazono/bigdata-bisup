# Разбор проекта

## Назначение

Этот раздел разбирает не абстрактный учебный стек, а конкретную реализацию репозитория `airflow-dwh-proj`.
Здесь важно смотреть на реальные файлы, реальные compose-конфигурации, реальные DAG-и и реальные SQL-операции.

## Главный принцип раздела

Если код, SQL или compose-файлы говорят одно, а поверхностное описание проекта говорит другое, приоритет всегда у кода и конфигурации.
Поэтому в этом блоке сознательно отмечаются и сильные стороны реализации, и учебные упрощения, и неполные части контура.
